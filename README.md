# 🛡️ 우리가족 금융가드 AI

문자나 사진 한 장이면, AI가 보이스피싱·투자사기 여부를 확인하고 **왜 위험한지 쉬운 말로 설명**해주는 웹 서비스입니다. 검사 결과는 카카오톡으로 바로 공유할 수 있는 경고 카드로도 만들어져, 혼자 판단하기 어려운 순간에 가족이 함께 개입할 수 있도록 돕습니다.

**[🔗 데모 바로 써보기](https://senior-financial-safety-guard-ai-63hyainpbjvwx3gpk9prbz.streamlit.app/)**

---

## 왜 만들었나

- 보이스피싱 피해자 중 **50대 이상 비중이 2023년 32% → 2025년 53%로** 계속 늘고 있습니다. (경찰청 국가수사본부)
- 검찰·경찰·금융감독원을 사칭하는 '기관사칭형' 범죄는 2016년 대비 발생 건수가 약 **4배** 늘어, 이제 전체 보이스피싱의 **51%를** 차지하는 최다 유형이 됐습니다. (경찰청 국가수사본부, 공공데이터포털)
- 디지털 기기에 익숙하지 않은 세대가 가장 큰 피해를 보고 있지만, 정작 이들을 위한 서비스는 대부분 "탐지 후 자동 차단"에 집중되어 있고, 왜 위험한지 스스로 이해하거나 가족이 함께 대응하도록 돕는 부분은 상대적으로 약하다고 판단했습니다.

## 무엇이 다른가

기존 보이스피싱 방지 서비스(시티즌코난, 피싱아이즈 등)는 스마트폰에 상주하며 악성 앱을 탐지하고 금융거래를 자동 차단하는 데 강점이 있는, 이미 검증된 서비스들입니다. 이 프로젝트는 그것을 대체하려는 것이 아니라, 그 서비스들이 상대적으로 약한 지점을 보완하는 데 초점을 맞췄습니다.

| | 시티즌코난 / 피싱아이즈 | 이 프로젝트 |
|---|---|---|
| 방식 | 백그라운드 자동 탐지·차단 | 의심되는 문자/사진을 직접 붙여넣어 확인 |
| 초점 | 악성 앱·통신 채널 차단 | 왜 사기인지 쉬운 말로 설명 |
| 가족 개입 | 별도 유료 멤버십 필요 | 진단 즉시 공유용 경고 카드 무료 자동 생성 |
| 설치 | 스마트폰 앱 설치 필요(안드로이드 전용) | 웹으로 바로 사용, 설치 불필요 |

## 주요 기능

- 텍스트 또는 사진(카톡·문자 캡처) 멀티모달 AI 분석 (Google Gemini API)
- 위험 / 주의 / 안전 3단계 판정 + 판단 근거 + 지금 해야 할 행동을 쉬운 말로 안내
- 가족·지인 공유용 경고 카드 자동 생성 + 원클릭 복사
- 실제 사기 유형 8종 데이터베이스 (경찰청·금융감독원 공개 통계 기반, 최신 발생 빈도순 정렬)
- 긴급 신고 원터치 안내 (경찰청 112 / 금융감독원 1332 / KISA 118)

## 실행 방법

### 1. 클론 및 패키지 설치

```bash
git clone <이 저장소 URL>
cd <저장소 폴더>
pip install -r requirements.txt
```

### 2. API 키 설정 (로컬 개발용)

프로젝트 루트에 `.streamlit/secrets.toml` 파일을 만들고 아래처럼 입력하세요. 이 파일은 `.gitignore`에 포함되어 있어 실수로 커밋되지 않습니다.

```toml
GEMINI_API_KEY = "여기에_본인의_Gemini_API_키"
```

API 키는 [Google AI Studio](https://aistudio.google.com/apikey)에서 무료로 발급받을 수 있습니다(무료 요금제로도 개인 데모 용도로는 충분합니다).

### 3. 실행

```bash
streamlit run app.py
```

앱 안에서도 "체험 모드"와 "내 API 키로 사용" 중 선택할 수 있습니다. secrets.toml에 키를 등록해두면 방문자가 별도 설정 없이 체험 모드(세션당 5회)로 바로 사용해볼 수 있습니다.

## 배포하기 (Streamlit Community Cloud, 무료)

1. 이 저장소를 본인 GitHub 계정으로 push 합니다.
2. [share.streamlit.io](https://share.streamlit.io) 접속 → GitHub 계정 연동 → 이 저장소 선택, 메인 파일로 `app.py` 지정
3. 배포 설정 화면의 **Advanced settings → Secrets**에 아래 내용을 추가합니다.
   ```toml
   GEMINI_API_KEY = "본인의_API_키"
   ```
4. Deploy를 누르면 몇 분 안에 `https://<앱이름>.streamlit.app` 형태의 공개 URL이 생성됩니다.

## 한계와 향후 계획

- **API 키 구조**: 현재는 배포자가 제공하는 무료 체험 키(세션당 5회 제한) 또는 사용자 본인의 키로 동작합니다. 실서비스가 되려면 서버가 키와 사용량·과금을 전담 관리하는 구조로 바뀌어야 합니다.
- **판단 정확도**: Gemini의 판단에 의존하는 구조라 오탐/미탐 가능성이 있습니다. 실서비스라면 경찰청·금융감독원의 실시간 사기 신고 DB와 교차 검증하는 구조가 필요합니다.
- **언어/대상 확장**: 현재는 한국어 텍스트/이미지만 지원합니다. 다문화가정, 외국인 노동자 등으로 사용 대상을 넓힐 여지가 있습니다.
- **데이터 활용 고도화**: 현재는 진단 시점의 단발성 분석만 이루어집니다. 향후에는 (개인정보를 제외한) 진단 이력을 익명으로 집계해 최근 유행하는 사기 유형 통계를 보여주는 기능을 추가할 계획입니다.
